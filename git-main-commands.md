## git clone
<a name="git-clone"></a>
Создает локальную копию удаленного репозитория.

**Синтаксис**

```git clone <path> ```

**Параметры**

`path` - путь к удаленному репозиторию.

**Пример**

```git clone ssh://git@b.yadro.com:7999/yt/techdocs.git```
